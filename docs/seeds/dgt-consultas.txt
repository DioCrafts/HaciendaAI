#  Cada línea no comentada es un número de consulta a ingerir vía
#  ingest-dgt.yml. Formato: V<NNNN>-<AA> (p.ej. V1234-24 = consulta nº 1234
#  resuelta en 2024).
#
#  IMPORTANTE: Los números de consulta se localizan en Petete:
#     https://petete.tributos.hacienda.gob.es/consultas/
#  Antes de activar una línea, VERIFICA el número en Petete. Los temas
#  marcados como "BÚSQUEDA SUGERIDA" describen el supuesto a buscar; el
#  operador localiza el número exacto y lo añade descomentado.
#
#  Recordatorio doctrinal (art. 89 LGT): las consultas vinculantes obligan
#  a la AEAT para el consultante y para terceros en supuestos idénticos,
#  pero NO vinculan a los tribunales. El LLM debe distinguir al citar.
